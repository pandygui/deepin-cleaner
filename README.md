# Deepin очиститель

системный очиститель инструменты.

## зависимости

* sudo apt install libdtkcore-dev libdtkwidget-dev

## Как запустить ?

* mkdir build

* cd build

* qmake ..

* make

## Применение

* ./deepin-cleaner
